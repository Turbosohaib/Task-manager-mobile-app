## Clockify Clone mobile app

Clone repo with:
```
git clone https://github.com/Turbosohaib/clockify-clone-mobile-app.git
```

Run:
 ```
 npm install
 ```
 Run app:
 ```
 npm run dev
 ```

 visit: 
 ```
 localhost:3000
 ```
